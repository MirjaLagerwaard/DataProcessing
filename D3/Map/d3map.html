Domestic Cat Population of Europe (1996 - 1998)









Source: http://pictures-of-cats.org/cat-population-as-a-percentage-of-human-population.html

I used the same data as for the previous map assignment, but this time I
have chosen a different color scheme with more contrasts. As mouseover
effects I enabled border color to change in darker grey, so the country is
highlighted. I disabled fill color to change on mouseover, because I did
not want to change the color scheme. I also made a popup on mouseover
which shows the name of the country and the data value (when available).
